# おやじストッパー / Oyaji Stopper

*おやじギャグは文化遺産。ツッコミは愛情。データは誇り。*

---

## 🎯 企画概要

おやじギャグ連発・昭和トーク無限ループ・絵文字洪水を**優しくツッコむ相棒AI**。  
深い感情介入はしない。**ユーモアだけで周囲の疲弊を防ぐ**。

日本の高文脈文化に最適化。**恥をかかせず、笑いで解決**。

このサービスは「自分のおやじ化を自覚している人」のためのツールです。  
矯正ではなく、おやじ化と**仲良くなるための装置**。

```
Before: 「またその話…（心の声）」
After:  AI「はい3回目～！」→ 一同笑い
```

---

## 🤔 なぜこれが必要か？

| 問題 | 従来の解決 | おやじストッパー |
|------|------------|-----------------|
| おやじギャグ連発 | 無視／我慢 | 「はい3回目～！」と軽くツッコミ |
| 昭和トークループ | 話題変更困難 | 「その話今週5回目です」と実況 |
| 絵文字洪水 | チャット崩壊 | 「絵文字祭りですね！」と称賛 |
| 本人無自覚 | 指摘しづらい | AIが相棒として自然に伝達 |

## 🌅 おやじデイリーリズム

おやじストッパーは1日を通じておやじ化と共存するリズムを提供します。

### 朝：おやじ占い
> 「今日のおやじ運：★★★☆☆　ダジャレが降ってきやすい一日です。ラッキーツッコミ：『はい3回目〜』」

占いなので外れても笑える。プレッシャーゼロ。

### 日中：リアルタイムツッコミ
場に応じたアバターが、タイミングよく（あるいは空気を読まずに）ツッコミを入れます。

### 夜：今日のおやじ総括
> 「今日のおやじ度：73ポイント。ダジャレ4回、昭和ワード12回。でも誰も傷つけなかったね。明日も元気におやじってください。」

---

## 🚀 主な機能

- **リアルタイムおやじ検知**　`ギャグ3連続 → 「はい3回目～！」`
- **昭和度メーター**　`キーワード密度75% → 「昭和度危険水域です！」`
- **場の自動検知**　`自宅／職場／飲み屋ごとにモード自動切替（手動上書き可）`
- **自動ギャグストック**　`ネタ切れ時 → 「次はこのダジャレどう？」`
- **P2Pギャグ交換**　`友達と新ネタ交換（AirDrop/Quick Share/WebRTC）`
- **アバター7種**　`冷徹秘書 / 若者後輩 / 昭和司会者 / 孫キャラ / おかん / ギャル後輩 / 飲み屋の友達`
- **落ち込み寄り添い**　`声トーン低下 → 「昭和話なら私が聞きますよ」`
- **おやじデイリーリズム**　`朝の占い → 日中ツッコミ → 夜の総括`
